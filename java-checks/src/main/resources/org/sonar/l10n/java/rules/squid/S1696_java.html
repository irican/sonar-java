应该避免 NullPointerException ，而不是捕获。显式捕获NullPointerException 的任何情况都可以很容易地转换为 null 测试，并且在catch块中执行的任何行为都可以轻松地移动到“is null”条件的分支。

不合规的代码示例

public int lengthPlus(String str) {
  int len = 2;
  try {
    len += str.length();
  }
  catch (NullPointerException e) {
    log.info("argument was null");
  }
  return len;
}

合规的解决方案

public int lengthPlus(String str) {
  int len = 2;

  if (str != null) {
    len += str.length();
  }
  else {
    log.info("argument was null");
  }
  return len;
}
