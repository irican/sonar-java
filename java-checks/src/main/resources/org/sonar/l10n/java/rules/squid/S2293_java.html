Java 7引入了菱形运算符(<>) 来减少泛型代码的冗长。 例如，您现在可以使用List简化构造函数声明，而不必在其声明及其构造函数中声明<>的类型， 编译器会推断出类型。

注意 当项目的sonar.java.source 低于7时，会自动禁用此规则。

不合规的代码示例

List<String> strings = new ArrayList<String>();  // 不合规的
Map<String,List<Integer>> map = new HashMap<String,List<Integer>>();  // 不合规的

合规解决方案

List<String> strings = new ArrayList<>();
Map<String,List<Integer>> map = new HashMap<>();
