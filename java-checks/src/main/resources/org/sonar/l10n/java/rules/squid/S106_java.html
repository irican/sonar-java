记录日志消息时，必须满足以下几个重要要求：

  • 用户必须能够轻松检索日志
  • 所有日志消息的格式必须统一，以便允许用户轻松读取日志
  • 日志数据必须确实被记录了
  • 敏感数据必须被安全的记录

如果日志数据被程序直接写入标准输出，将无法满足上述要求，所以强烈建议定义和使用专用的记录器（Logger）。

不合规的代码示例

System.out.println("My Message");  // 不合规

合规的解决方案

logger.log("My Message");
