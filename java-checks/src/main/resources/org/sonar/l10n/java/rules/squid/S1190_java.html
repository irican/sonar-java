添加了一些 Java更新的关键字。 虽然使用这些单词作为标识符的代码可以在旧版本的Java下编译，但在现代版本下就不能编译了。

以下关键字被标记为无效标识符

Keyword Added
_ 9
enum 5.0

assert 和 strictfp 是有效标识符的另一个示例，它在以后的版本中成为关键字，但是如SONARJAVA-285中所述， 支持解析代码并不容易。 这些旧版本因此不受此规则支持。

不合规的代码示例

public void doSomething() {
  int enum = 42;            // 不合规
  String _ = "";   // 不合规
}

合规的解决方案

public void doSomething() {
  int magic = 42;
}
