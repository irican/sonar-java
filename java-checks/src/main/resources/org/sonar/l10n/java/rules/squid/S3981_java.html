集合的大小和数组的长度始终大于或等于零。 所以测试大小或长度是否大于或 等于零没有意义，因为结果总是 true 。 同样测试它小于零将始终返回 false。 也许意图是检查集合或数组的非空虚。

不符合规范的代码示例

if (myList.size() >= 0) { ... }

if (myList.size() < 0) { ... }

boolean result = myArray.length >= 0;

if (0 > myArray.length) { ... }

合规的解决方案

if (!myList.isEmpty()) { ... }

if (myArray.length >= 42) { ... }
