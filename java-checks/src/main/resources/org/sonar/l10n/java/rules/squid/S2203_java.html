目前来说，虽然您可以使用 forEach(list::add)或者 collect和Stream, collect 是更好的选择，因为它是自动线程安全和可平行的。

不合规的代码示例

List<String> bookNames = new ArrayList<>();
books.stream().filter(book -> book.getIsbn().startsWith("0"))
                .map(Book::getTitle)
                .forEach(bookNames::add);  // 不合规的

合规解决方案

List<String> bookNames = books.stream().filter(book -> book.getIsbn().startsWith("0"))
                .map(Book::getTitle)
                .collect(Collectors.toList());
