始终返回 true 或 false 的 instanceof 运算符要么无用，要么是误解的结果，这可能导致生产中的意外行为。

不合规的代码示例

public boolean isSuitable(Integer param) {
...
  String name = null;

  if (name instanceof String) { // 不合规； 始终为false，因为name为null
    //...
  }

  if(param instanceof Number) {  // 不合规； 除非param为null，否则总是为true，因为param是一个Integer
    doSomething();
  }
...
}

合规的解决方案

public boolean isSuitable(Integer param) {
...
  doSomething();
...
}
