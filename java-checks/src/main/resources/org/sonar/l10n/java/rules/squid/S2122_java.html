java.util.concurrent.ScheduledThreadPoolExecutor 的池的大小取决于 corePoolSize，因此将 corePoolSize 设置为零意味着执行器将没有线程，并且什么也不运行。

此规则通过其setter或对象构造函数检测 corePoolSize 设置为零的实例。
