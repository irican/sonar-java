方法没有方法体有以下几个原因:

  • 这是一种无意的遗漏，应该加以纠正，以防止生产中出现意外行为。
  • 它尚未被支持或永远不会支持。在这种情况下，应抛出 UnsupportedOperationException 。
  • 该方法是一个有意为空的重写。在这种情况下，嵌套注释应该解释空白重写的原因。

不合规的代码示例

public void doSomething() {
}

public void doSomethingElse() {
}

合规的解决方案

@Override
public void doSomething() {
  // 因为X和Y，什么都不做。
}

@Override
public void doSomethingElse() {
  throw new UnsupportedOperationException();
}

例外

当类中有其他构造函数时，将忽略缺省（无参数）构造函数，抽象类中的空方法也是如此。

public abstract class Animal {
  void speak() {  // 缺省实现被忽略
  }
}
