根据官方的javadoc文档，当垃圾收集（garbage collection）确定不再引用该对象时， 此对象上的垃圾收集器（garbage collector）会调用此Object.finalize（）。显式地调用该方法会破坏此契约，因此具有误导性。

不合规的代码示例

public void dispose() throws Throwable {
  this.finalize();                       // 不合规
}

请参阅

  • MITRE, CWE-586 - Explicit Call to Finalize()
  • CERT, MET12-J. - Do not use finalizers
