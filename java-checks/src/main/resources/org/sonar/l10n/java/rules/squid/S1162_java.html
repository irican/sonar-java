检查异常的目的是确保通过传播或处理错误来解决错误，但是有些人认为检查异常会将错误处理/传播逻辑传播到任何地方， 从而对源代码的可读性产生负面影响。

此规则验证没有方法抛出新的已检查异常。

不合规的代码示例

public void myMethod1() throws CheckedException {
  ...
  throw new CheckedException(message);   // 不合规
  ...
  throw new IllegalArgumentException(message); // 合规； IllegalArgumentException 未检查
}

public void myMethod2() throws CheckedException {  // 合规； 传播被允许
  myMethod1();
}
