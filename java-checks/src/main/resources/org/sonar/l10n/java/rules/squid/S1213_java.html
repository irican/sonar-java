根据Oracle定义的Java代码约定，接口声明或类的成员应该在源文件中按如下顺序出现:

  • 类和实例变量
  • 构造函数
  • 方法

不合规的代码示例

public class Foo{
   private int field = 0;
   public boolean isTrue() {...}
   public Foo() {...}                         // 不合规， 方法后定义了构造函数
   public static final int OPEN = 4;  //不合规，构造函数和方法之后定义了变量
}

合规的解决方案

public class Foo{
   public static final int OPEN = 4;
   private int field = 0;
   public Foo() {...}
   public boolean isTrue() {...}
}
