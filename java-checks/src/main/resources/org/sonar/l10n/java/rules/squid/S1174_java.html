Object.finalize() 方法的契约很明确：只有垃圾收集器（Garbage Collector）才能调用此方法。

公开此方法会产生误导，因为这意味着任何调用者都可以使用它。

不合规的代码示例

public class MyClass {

  @Override
  public void finalize() {    // 不合规
    /* ... */
  }
}

请参阅

  • MITRE, CWE-583 - finalize() Method Declared Public
  • CERT, MET12-J. - Do not use finalizers
