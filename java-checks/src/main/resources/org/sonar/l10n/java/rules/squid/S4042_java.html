当 java.io.File＃delete 失败时，此 boolean 方法只返回 false ，但没有指示原因。 上 另一方面，当 java.nio.Files＃delete 失败时，此 void 方法返回一系列异常类型之一，以更好地指出失败的原因。 并且由于在调试情况下通常更多信息更好， java.nio.Files＃delete 是首选选项。

不符合规范的代码示例

public void cleanUp(Path path) {
  File file = new File(path);
  if (!file.delete()) {  // 不合规的
    //...
  }
}

合规解决方案

public void cleanUp(Path path) throws NoSuchFileException, DirectoryNotEmptyException, IOException{
  Files.delete(path);
}
