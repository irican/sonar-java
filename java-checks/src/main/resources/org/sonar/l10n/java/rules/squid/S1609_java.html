单抽象（Single Abstract）方法（SAM）接口是仅包含一个方法的Java接口。 Java API充满了SAM接口，例如 java.lang.Runnable， java.awt.event.ActionListener， java.util.Comparator 和 java.util.concurrent.Callable。SAM接口在Java 8中有一个特殊的地位， 因为它们可以使用Lambda表达式或Method引用来实现。

使用 @FunctionalInterface 在向SAM添加一个额外的非重写抽象方法时强制执行编译中断，这将中断Lambda实现的使用。

注意 当项目的 sonar.java.source 低于 8 时，会自动禁用此规则。

不合规的代码示例

public interface Changeable<T> {
  public void change(T o);
}

合规的解决方案

@FunctionalInterface
public interface Changeable<T> {
  public void change(T o);
}

弃用

此规则已弃用，最终将被删除。
