重写父类方法会阻止调用该方法，除非在重写方法中进行明确的 super 调用。在某些情况下，不能调用 super 方法，但不能使用JUnit 3 TestCase 中的 setUp 和 tearDown

不合规的代码示例

public class MyClassTest extends MyAbstractTestCase {

  private MyClass myClass;
    @Override
    protected void setUp() throws Exception {  // 不合规的
      myClass = new MyClass();
    }

合规的解决方案

public class MyClassTest extends MyAbstractTestCase {

  private MyClass myClass;
    @Override
    protected void setUp() throws Exception {
      super.setUp();
      myClass = new MyClass();
    }
