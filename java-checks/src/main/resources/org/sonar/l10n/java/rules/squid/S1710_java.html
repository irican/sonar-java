在Java 8之前，如果需要使用同一个注解（annotation）的多个实例，就必须将它们封装在容器注释中。在Java 8中，这不再是必需的，允许更干净、更可读的代码。

注意 当项目的 sonar.java.source 低于 8时，会自动禁用此规则。

不合规的代码示例

@SomeAnnotations({  // 不合规
  @SomeAnnotation(..a..),
  @SomeAnnotation(..b..),
  @SomeAnnotation(..c..),
})
public class SomeClass {
  ...
}

合规的解决方案

@SomeAnnotation(..a..)
@SomeAnnotation(..b..)
@SomeAnnotation(..c..)
public class SomeClass {
  ...
}
