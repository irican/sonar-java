嵌套 if， for， while， switch， 和 try 语句是制作所谓“意大利面条代码（Spaghetti code）”的关键因素。

这样的代码很难读取、重构，因此很难维护。

不合规的代码示例

默认阈值为3：

if (condition1) {                  // 合规 - 深度 = 1
  /* ... */
  if (condition2) {                // 合规 - 深度 = 2
    /* ... */
    for(int i = 0; i < 10; i++) {  // 合规 - 深度 = 3， 不超过限制
      /* ... */
      if (condition4) {            // 不合规 - 深度 = 4
        if (condition5) {          // 深度 = 5， 超过限制，但问题仅在深度 = 4时报告
          /* ... */
        }
        return;
      }
    }
  }
}
