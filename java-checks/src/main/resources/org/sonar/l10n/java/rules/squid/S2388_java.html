当内部类扩展另一个类，并且它的外部类和它的父类都有一个具有相同名称的方法时，对该方法的调用可能会令人困惑。 编译器将解析对超类方法的调用，但维护者可能会感到困惑，因此应使用super.显式调用超类方法。

不合规的代码示例

public class Parent {
  public void foo() { ... }
}

public class Outer {

  public void foo() { ... }

  public class Inner extends Parent {

    public void doTheThing() {
      foo();  // 不符合要求；是否其实应该为Outer.this.foo（）？ 
      // ...
    }
  }
}

合规解决方案

public class Parent {
  public void foo() { ... }
}

public class Outer {

  public void foo() { ... }

  public class Inner extends Parent {

    public void doTheThing() {
      super.foo();
      // ...
    }
  }
}
