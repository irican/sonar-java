标签（Labels）在Java中并不常用，许多开发人员不了解它们的工作原理。而且，它们的使用使得控制流更难以遵循，这降低了代码的可读性。

不合规的代码示例

int matrix[][] = {
  {1, 2, 3},
  {4, 5, 6},
  {7, 8, 9}
};

outer: for (int row = 0; row < matrix.length; row++) {   // 不合规
  for (int col = 0; col < matrix[row].length; col++) {
    if (col == row) {
      continue outer;
    }
    System.out.println(matrix[row][col]);                // 打印对角线下的元素，即4,7和8
  }
}

合规的解决方案

for (int row = 1; row < matrix.length; row++) {          // 合规

  for (int col = 0; col < row; col++) {
    System.out.println(matrix[row][col]);                // 也会打印出 4， 7 和 8
  }
}
