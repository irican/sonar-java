Thread.run() 方法的目的是在一个单独的专用线程中执行代码。直接调用这个方法没有意义，因为它会导致其代码在当前线程中执行。

要获得预期的行为，可以调用 Thread.start() 方法来代替。

不合规的代码示例

Thread myThread = new Thread(runnable);
myThread.run(); // 不合规

合规的解决方案 

Thread myThread = new Thread(runnable);
myThread.start(); // 合规

请参阅

  • MITRE, CWE-572 - Call to Thread run() instead of start()
  • CERT THI00-J. - Do not invoke Thread.run()
