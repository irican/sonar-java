可选值可以包含值，也可以不包含值。可以使用以下命令访问 Optional 中保存的值 get（）方法，但它会抛出一个

NoSuchElementException 如果没有值存在。为避免异常，应始终调用 isPresent（）方法 在任何对 get（）的调用之前完成。

或者，请注意其他方法，例如 orElse（...）， orElseGet（...）或 orElseThrow（...）可用于指定如何处理空可选。

不符合规范的代码示例

Optional<String> value = this.getOptionalValue();

// ...

String stringValue = value.get(); // 不合规的

合规解决方案

Optional<String> value = this.getOptionalValue();

// ...

if (value.isPresent()) {
  String stringValue = value.get();
}

or

Optional<String> value = this.getOptionalValue();

// ...

String stringValue = value.orElse("default");
