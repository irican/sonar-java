当私有字段的值总是在读取之前分配给类的方法时，则不会使用它来存储类信息。因此，它应该成为相关方法中的局部变量，以防止任何误解。

不合规的代码示例

public class Foo {
  private int a;
  private int b;

  public void doSomething(int y) {
    a = y + 5;
    ...
    if(a == 0) {
      ...
    }
    ...
  }

  public void doSomethingElse(int y) {
    b = y + 3;
    ...
  }
}

合规的解决方法

public class Foo {

  public void doSomething(int y) {
    int a = y + 5;
    ...
    if(a == 0) {
      ...
    }
  }

  public void doSomethingElse(int y) {
    int b = y + 3;
    ...
  }
}

例外

此规则不会在带注释的字段上报告任何问题。
