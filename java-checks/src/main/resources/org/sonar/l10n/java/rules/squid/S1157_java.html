使用 toLowerCase() 或 toUpperCase() 进行不区分大小写的比较是低效的，因为它需要创建临时的中间（intermediate）String 对象。

不合规的代码示例

boolean result1 = foo.toUpperCase().equals(bar);             // 不合规
boolean result2 = foo.equals(bar.toUpperCase());             // 不合规
boolean result3 = foo.toLowerCase().equals(bar.LowerCase()); // 不合规

合规的解决方案

boolean result = foo.equalsIgnoreCase(bar);                  // 合规
