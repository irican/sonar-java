带有向错误方向移动的计数器的for（循环）不是无限循环。 由于环绕，循环最终将达到其停止条件，但在这样做时，它将比预期运行许多次，可能导致意外行为。

不符合规范的代码示例

public void doSomething(String [] strings) {
  for (int i = 0; i < strings.length; i--) { //不合规的;
    String string = strings[i];  // ArrayIndexOutOfBoundsException 当 i 达到 -1 时
    //...
  }

合规解决方案

public void doSomething(String [] strings) {
  for (int i = 0; i < strings.length; i++) {
    String string = strings[i];
    //...
  }
