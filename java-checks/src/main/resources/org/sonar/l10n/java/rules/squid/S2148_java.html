从Java 7开始，可以在数字文本中添加下划线('_')来增强可读性。以这种方式添加下划线没有语义意义，但使维护人员更容易理解代码。

触发此规则所需的小数点左侧的位数因基数而异。

基数 最小位数
二进制 9
八进制 9
十进制 6
十六进制 9

这条规则只检查下划线的存在，而不检查它们的间距。

注意 当项目的 sonar.java.source 低于 7 时，会自动禁用此规则。

不合规的代码示例

int i = 10000000;  // 不合规： 这是1000万还是1亿？
int  j = 0b01101001010011011110010101011110;  // 不合规
long l = 0x7fffffffffffffffL;  // 不合规

合规的解决方案

int i = 10_000_000;
int  j = 0b01101001_01001101_11100101_01011110;
long l = 0x7fff_ffff_ffff_ffffL;
