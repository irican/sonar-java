通过以下代码可以实现从指定偏移量开始查找给定子字符串： str.substring（beginIndex）.indexOf（char1）。 这很好用，但它为每次调用 substring 方法创建一个新的 String 。当这是在循环中完成时，很多 String是创建后没有用的，如果 str 很大，可能会导致性能问题。

为避免性能问题， String.substring（beginIndex）不应使用以下方法链接：

- indexOf(int ch)

- indexOf(String str)

- lastIndexOf(int ch)

- lastIndexOf(String str)

- startsWith(String prefix)

对于这些方法中的每一种，可以使用另一个带附加参数的方法来指定偏移量。

使用这些方法可以获得相同的结果，同时避免创建其他 String 实例。

不符合规范的代码示例

str.substring（的beginIndex）.indexOf（CHAR1）; //不合规;一个新的String将由“substring”创建


合规解决方案


str.indexOf(char1, beginIndex);
