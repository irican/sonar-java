当为局部变量分配一个未被任何后续指令读取的值时，就会发生死存储（dead store）。计算或检索一个值只是为了重写（overwrite）它或抛出它， 可能表明代码中存在严重错误。即使这不是一个错误，它也是严重的浪费资源。因此，应使用所有计算值。

不合规的代码示例

i = a + b; // 不合规；值被重写（overwrite）前未使用的计算结果
i = compute();

合规的解决方案

i = a + b;
i += compute();

例外

此规则忽略对 -1， 0， 1， null， true， false 和 "" 的初始化。

请参阅

  • MITRE, CWE-563 - Assignment to Variable without Use ('Unused Variable')
  • CERT, MSC13-C. - Detect and remove unused values
  • CERT, MSC56-J. - Detect and remove superfluous code and values
