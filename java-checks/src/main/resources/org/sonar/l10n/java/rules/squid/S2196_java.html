从Java 7开始， String可以用作 switch 参数。因此，当 if/else if 结构中的 String 针对三个或更多值进行测试时，应将其转换为 switch 以提高可读性。

注意当项目的 sonar.java.source 低于 7 时，会自动禁用此规则。

不合规的代码示例

if ("red".equals(choice)) {  // 不合规
  dispenseRed();
} else if ("blue".equals(choice)) {
  dispenseBlue();
} else if ("yellow".equals(choice)) {
  dispenseYellow();
} else {
  promptUser();
}

合规的解决方案

switch(choice) {
  case "Red":
    dispenseRed();
    break;
  case "Blue":
    dispenseBlue():
    break;
  case "Yellow":
    dispenseYellow();
    break;
  default:
    promptUser();
    break;
}
