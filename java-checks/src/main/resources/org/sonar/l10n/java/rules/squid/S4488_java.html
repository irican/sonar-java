Spring framework 4.3引入了 @RequestMapping 注释的变体，以更好地表示带注释的方法的语义。 使用 @GetMapping, @PostMapping, @PutMapping, @PatchMapping 和 @DeleteMapping 前者的使用应优先于使用 @RequestMapping(method = RequestMethod.XYZ).

不符合规范的代码示例

@RequestMapping(path = "/greeting", method = RequestMethod.GET) // 不合规的
public Greeting greeting(@RequestParam(value = "name", defaultValue = "World") String name) {
...
}

合规解决方案

@GetMapping(path = "/greeting") // 合规的
public Greeting greeting(@RequestParam(value = "name", defaultValue = "World") String name) {
...
}
