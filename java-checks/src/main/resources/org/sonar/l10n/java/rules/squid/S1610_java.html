使用Java 8的“默认方法（default method）”功能，任何没有直接或继承字段的抽象类都应转换为接口。但是，此更改可能不适用于要将该类用作API的库或其他应用程序。

注意 当项目的 sonar.java.source 低于 8 时，会自动禁用此规则。

不合规的代码示例

public abstract class Car {
  public abstract void start(Environment c);

  public void stop(Environment c) {
    c.freeze(this);
  }
}

合规的解决方案

public interface Car {
  public void start(Environment c);

  public default void stop(Environment c) {
    c.freeze(this);
  }
}
