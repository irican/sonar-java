具有至多一次迭代的循环等效于使用if语句来有条件地执行一段代码。没有开发人员希望找到循环语句的这种用法。 如果作者的最初意图确实是有条件地执行一段代码，则应该使用if语句。

在最坏的情况下，这并不是作者最初的意图，所以循环体应该被修复，以更合适的方式使用嵌套的 return，break 或 throw 语句。

不合规的代码示例

for (int i = 0; i < 10; i++) { // 不合规，循环只执行了一次
  printf("i is %d", i);
  break;
}
...
for (int i = 0; i < 10; i++) { // 不合规，循环只执行了一次
  if(i == x) {
    break;
  } else {
    printf("i is %d", i);
    return;
  }
}

合规的解决方案

for (int i = 0; i < 10; i++) {
  printf("i is %d", i);
}
...
for (int i = 0; i < 10; i++) {
  if(i == x) {
    break;
  } else {
    printf("i is %d", i);
  }
}
