魔法数字（magic number）是一个无处不在的数字，直接用在一个语句中。通常使用魔法数字，例如限制循环的迭代次数，测试属性的值等。

当你在写一段代码时，使用魔法数字可能看起来很明显和直接，但在调试时它们却不是如此。

所以说在使用之前，必须先将魔法数字赋给明确命名的变量，从而使魔法数字有清晰的意义。

-1、0和1不被当做是魔法数字。

不合规的代码示例

public static void doSomething() {
	for(int i = 0; i < 4; i++){                 // 不合规，4是魔法数字
		...
	}
}

合规的解决方案

public static final int NUMBER_OF_CYCLES = 4;
public static void doSomething() {
  for(int i = 0; i < NUMBER_OF_CYCLES ; i++){
    ...
  }
}

例外

这个规则忽略了 hashCode 方法
