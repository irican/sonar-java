当一个private方法仅由内部类调用时，没有理由不将它移动到该类中。它仍将具有对外部类成员的相同访问权限，但外部类将更清晰，更简洁。

不合规的解决方案

public class Outie {
  private int i=0;

  private void increment() {  //不合规的
    i++;
  }

  public class Innie {
    public void doTheThing() {
      Outie.this.increment();
    }
  }
}

合规的解决方案

public class Outie {
  private int i=0;

  public class Innie {
    public void doTheThing() {
      Outie.this.increment();
    }

    private void increment() {
      Outie.this.i++;
    }
  }
}
