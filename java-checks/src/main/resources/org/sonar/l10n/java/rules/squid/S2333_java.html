interface中声明的方法默认为 public 和 abstract 任何变量都是自动的 public static final.没有必要明确声明它们。

由于注释是隐式接口，因此它们也适用。

类似地， final 修饰符在final 类的任何方法上都是多余的，并且private 在Enum的构造函数上是多余的。

不合规的代码示例

public interface Vehicle {

  public void go(int speed, Direction direction);  // 不合规的

合规解决方案

public interface Vehicle {

  void go(int speed, Direction direction);
