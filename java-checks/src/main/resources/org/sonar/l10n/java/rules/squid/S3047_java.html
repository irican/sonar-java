当一个方法对同一组数据进行了多次循环时，无论这是一个列表还是一组数字，将这些循环整合在一起都很可能提高该方法的效率。

不合规的代码示例

public void doSomethingToAList(List<String> strings) {
  for (String str : strings) {
    doStep1(str);
  }
  for (String str : strings) {  // 不合规
    doStep2(str);
  }
}

合规的解决方案

public void doSomethingToAList(List<String> strings) {
  for (String str : strings) {
    doStep1(str);
    doStep2(str);
  }
}
