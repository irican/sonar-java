Files.exists 方法在JDK 8中的性能明显较差，并且在用于检查实际不存在的文件时会显着减慢应用程序的速度。

Files.notExists ， Files.isDirectory 和 Files.isRegularFile 也是如此

注意当项目的 sonar.java.source 不是8时，会自动禁用此规则。

不符合规范的代码示例

Path myPath;
if(java.nio.Files.exists(myPath)) {  //不符合规范的
 // do something（一些代码）
}

合规的解决方案

Path myPath;
if(myPath.toFile().exists())) {
 // do something（一些代码）
}
