必须谨慎地重写 Object.finalize() 方法，以处理一些系统资源。

如果父实现（parent implementations）还必须部署一些系统资源，强烈建议在此方法实现结束时调用 super.finalize() 。

不合规的代码示例

protected void finalize() {   // 不合规；没有调用 super.finalize();
  releaseSomeResources();
}

protected void finalize() {
  super.finalize();  // 不合规；这个调用应放在最后
  releaseSomeResources();
}

合规的解决方案

protected void finalize() {
  releaseSomeResources();
  super.finalize();
}

  • CERT, MET12-J. - Do not use finalizers
