默认拦截器，如应用程序安全拦截器，必须列在 ejb-jar.xml 文件中，否则它们不会被视为默认的。

此规则应用于包含JEE Bean的项目（javax.ejb.Singleton，MessageDriven，Stateless 或 Stateful 中的任意一个）。

不合规的代码示例

// file: ejb-interceptors.xml
<assembly-descriptor>
 <interceptor-binding> <!-- should be declared in ejb-jar.xml -->
      <ejb-name>*</ejb-name>
      <interceptor-class>com.myco.ImportantInterceptor</interceptor-class><!-- Noncompliant; will NOT be treated as default -->
   </interceptor-binding>
</assembly-descriptor>

Compliant Solution

// file: ejb-jar.xml
<assembly-descriptor>
 <interceptor-binding>
      <ejb-name>*</ejb-name>
      <interceptor-class>com.myco.ImportantInterceptor</interceptor-class>
   </interceptor-binding>
</assembly-descriptor>
