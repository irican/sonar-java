调用 System.exit(int status) 或 Rutime.getRuntime().exit(int status) 会调用 shutdown hooks 并关闭整个Java虚拟机。 而调用 Runtime.getRuntime().halt(int) 会立即关闭，而且不会调用 shutdown hooks，也不会跳过 finalization。

这些方法中的每一个都应该非常小心地使用，并且只有在意图停止整个Java进程时才使用。例如，不应该从运行在J2EE容器中的应用程序中调用它们。

不合规的代码示例

System.exit(0);
Runtime.getRuntime().exit(0);
Runtime.getRuntime().halt(0);

例外

在 main 中忽略这些方法。

请参阅

  • MITRE, CWE-382 - Use of System.exit()
  • CERT, ERR09-J. - Do not allow untrusted code to terminate the JVM
