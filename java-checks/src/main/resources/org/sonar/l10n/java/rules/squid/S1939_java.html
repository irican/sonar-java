所有类都隐式地extend Object 。显式地这样做是多余的。

此外，如果其父项也是冗余的，则声明接口 and 的实现。如果您实现了接口，您还将隐式地实现它的父类，而不需要显式地实现。

不合规的代码示例

public interface MyFace {
  // ...
}

public interface MyOtherFace extends MyFace {
  // ...
}

public class Foo
    extends Object // Noncompliant
    implements MyFace, MyOtherFace {  // 不合规
  //...
}

合规的解决方案

public interface MyFace {
  // ...
}

public interface MyOtherFace extends MyFace {
  // ...
}

public class Foo implements MyOtherFace {
  //...
}
