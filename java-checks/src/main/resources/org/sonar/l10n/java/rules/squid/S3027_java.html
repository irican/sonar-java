使用单个字母String的indexOf或lastIndexOf 调用可以通过切换到使用char参数的调用来提高性能。

不合规的代码示例

String myStr = "Hello World";
// ...
int pos = myStr.indexOf("W");  // 不合规
// ...
int otherPos = myStr.lastIndexOf("r"); // 不合规
// ...

合规的解决方案

String myStr = "Hello World";
// ...
int pos = myStr.indexOf('W');
// ...
int otherPos = myStr.lastIndexOf('r');
// ...
