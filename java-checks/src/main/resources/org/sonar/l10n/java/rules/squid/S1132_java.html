最好将字符串文本放在 equals() 或 equalsIgnoreCase() 方法调用的左侧。

这可以防止引发空指针异常，因为根据定义，字符串文字永远不能为null。

不合规的代码示例

String myString = null;

System.out.println("Equal? " + myString.equals("foo"));                        // 不合规；将提出 NPE
System.out.println("Equal? " + (myString != null && myString.equals("foo")));  // 不合规； 可以删除null检查

合规的解决方案

System.out.println("Equal?" + "foo".equals(myString));                         // 恰当的处理为null的情况
