如果没有给出任何参数，Collections.toArray 方法返回一个Object []，如果尝试将其转换为适当类的数组，它将在运行时导致 ClassCastException。相反，应将正确类型的数组传递给调用。

不合规的代码示例

public String [] getStringArray(List<String> strings) {
  return (String []) strings.toArray();  // 不合规的；抛出ClassCastException
}

合规的解决方案

public String [] getStringArray(List<String> strings) {
  return strings.toArray(new String[0]);
}
