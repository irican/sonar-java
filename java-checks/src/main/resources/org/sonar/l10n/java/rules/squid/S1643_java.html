字符串是不可变的对象，所以连接不只是将新字符串添加到现有字符串的末尾。相反，在每个循环迭代中，将第一个字符串转换为中间对象类型， 添加第二个字符串，然后将中间对象转换回字符串。此外，随着字符串变长，这些中间操作的性能会下降。因此，最好使用StringBuilder。

不合规的代码示例

String str = "";
for (int i = 0; i < arrayOfStrings.length ; ++i) {
  str = str + arrayOfStrings[i];
}

合规的解决方案

StringBuilder bld = new StringBuilder();
  for (int i = 0; i < arrayOfStrings.length; ++i) {
    bld.append(arrayOfStrings[i]);
  }
  String str = bld.toString();
