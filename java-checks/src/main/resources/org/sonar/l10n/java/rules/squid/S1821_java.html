嵌套的 switch 结构很难理解，因为您很容易将内部 switch 的case混淆为属于外部的语句。 因此，应避免使用嵌套的 switch 语句。

具体来说，您应该构造代码，以避免嵌套 switch 语句，但是如果不能，则考虑将内部 switch 转移到另一个函数。

不合规的代码示例

void foo(int n, int m) {
  switch (n) {
    case 0:
      switch (m) {  // 不合规； 嵌套的 switch
        // ...
      }
    case 1:
      // ...
    default:
      // ...
  }
}

合规的解决方案

void foo(int n, int m) {
  switch (n) {
    case 0:
      bar(m);
    case 1:
      // ...
    default:
      // ...
  }
}

void bar(int m){
  switch(m) {
    // ...
  }
}
