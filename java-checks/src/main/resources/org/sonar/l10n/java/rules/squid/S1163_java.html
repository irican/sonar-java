从finally块中抛出异常将掩盖（mask）先前在 try 或 catch 块中抛出的异常，并且掩盖的异常消息和堆栈跟踪将丢失。

不合规的代码示例

try {
  /* some work which end up throwing an exception */
  throw new IllegalArgumentException();
} finally {
  /* clean up */
  throw new RuntimeException();       // 不合规； 掩盖了 IllegalArgumentException
}

合规的解决方案

try {
  /* some work which end up throwing an exception */
  throw new IllegalArgumentException();
} finally {
  /* clean up */
}

请参阅

  • CERT, ERR05-J. - Do not let checked exceptions escape from a finally block
