Java的 import 机制允许使用简单的类名。因此，在 import 的文件中使用类的完全限定名（fully qualified name）是多余的， 而且容易混淆。

不合规的代码示例

import java.util.List;
import java.sql.Timestamp;

//...

java.util.List<String> myList;  // 不合规
java.sql.Timestamp tStamp; // 不合规

合规的解决方案

import java.util.List;
import java.sql.Timestamp;

//...

List<String> myList;
Timestamp tStamp;
