根据Java语言规范：

为了与旧版本的Java SE平台兼容，

返回数组的方法的声明允许放置（部分或全部）空括号对，这些空括号对在形式参数列表之后形成数组类型的声明。

不应在新代码中使用此过时语法。

不合规的代码示例

public int getVector()[] { /* ... */ }    // 不合规

public int[] getMatrix()[] { /* ... */ }  // 不合规

合规的解决方案

public int[] getVector() { /* ... */ }

public int[][] getMatrix() { /* ... */ }
