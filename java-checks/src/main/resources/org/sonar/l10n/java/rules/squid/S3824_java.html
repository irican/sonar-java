在继续添加或更改地图中的值之前，对 null 测试 java.util.Map.get（）的结果是一种常见模式。 但是 java.util.Map API以 computeIfPresent（）和 computeIfAbsent（）方法的形式提供了一个明显更好的选择。 使用这些代码可以使代码更清晰，更易读。

注意当项目的 sonar.java.source 不是8时，会自动禁用此规则。

不符合规范的代码示例

V value = map.get(key);
if (value == null) {  // 不符合规范的
  value = V.createFor(key);
  if (value != null) {
    map.put(key, value);
  }
}
return value;

合规的解决方案

return map.computeIfAbsent(key, k -> V.createFor(k));
