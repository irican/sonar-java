调用hashCode 可以返回 Integer.MIN_VALUE。取这样一个hashcode的绝对值，仍然会得到一个负数。因为你的代码可能会假设它是一个正值，所以结果将是不可靠的。

类似地，Integer.MIN_VALUE可以从Random.nextInt() 或任何对象的compareTo方法，Long.MIN_VALUE 可以从 Random.nextLong()返回。 从这些方法返回的值调用Math.abs也是不明智的。

不合规的代码示例

public void doSomething(String str) {
  if (Math.abs(str.hashCode()) > 0) { // 不合规
    // ...
  }
}

合规的解决方案

public void doSomething(String str) {
  if (str.hashCode() != 0) {
    // ...
  }
}
