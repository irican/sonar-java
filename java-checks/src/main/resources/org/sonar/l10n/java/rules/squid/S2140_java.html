无需将 Random 的 nextDouble 方法的输出相乘以获得随机整数。请改用 nextInt 方法。

当返回浮点值的任何 Random方法的返回值转换为整数时，此规则会报告问题。

不合规的代码示例

Random r = new Random();
int rand = (int)r.nextDouble() * 50;  // 获取0到50之间的伪随机值的不合规方法
int rand2 = (int)r.nextFloat(); // 不合规； 将总是 0;

合规的解决方案

Random r = new Random();
int rand = r.nextInt(50);  // 返回0到50之间的伪随机值
