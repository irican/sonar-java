enum 通常被认为是常量，但 enum 如果有 public 字段或 public 型的 setter方法就不仅不是常量，还容易受到恶意代码的攻击。理想情况下，enum 里的字段应是 private 型，并且 在构造函数中进行设置。但如果这样的方式不可行，就应尽可能的降低它们的可见性。

不合规的代码示例

public enum Continent {

  NORTH_AMERICA (23, 24709000),
  // ……
  EUROPE (50, 39310000);

  public int countryCount;  // 不合规
  private int landMass;

  Continent(int countryCount, int landMass) {
    // ……
  }

  public void setLandMass(int landMass) {  // 不合规
    this.landMass = landMass;
  }

合规的解决方案

public enum Continent {

  NORTH_AMERICA (23, 24709000),
  // ……
  EUROPE (50, 39310000);

  private int countryCount;
  private int landMass;

  Continent(int countryCount, int landMass) {
    // ……
  }
