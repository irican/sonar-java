notify 和 notifyAll 都可以唤醒睡眠线程，但是notify只可以唤醒一个，而notifyAll可以唤醒所有。 因此notify可能无法唤醒正确的线程，应该使用notifyAll。

不合规的代码示例

class MyThread extends Thread{

  @Override
  public void run(){
    synchronized(this){
      // ...
      notify();  // 不合规
    }
  }
}

合规的解决方案

class MyThread extends Thread{

  @Override
  public void run(){
    synchronized(this){
      // ...
      notifyAll();
    }
  }
}
