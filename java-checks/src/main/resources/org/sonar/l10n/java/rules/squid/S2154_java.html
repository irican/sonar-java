如果在三元运算符（例如 a?b:c）中使用封装的原始类型值 （例如 Integers 和 Floats） 则两个值都将被拆箱并强制转换为公共类型（common type），可能会导致意外结果。要避免这种情况，请向兼容类型添加显式强制转换。

不合规的代码示例

Integer i = 123456789;
Float f = 1.0f;
Number n = condition ? i : f;  // 不合规； i 强制转换为float类型。 n = 1.23456792E8

合规的解决方案

Integer i = 123456789;
Float f = 1.0f;
Number n = condition ? (Number) i : f;  // n = 123456789
