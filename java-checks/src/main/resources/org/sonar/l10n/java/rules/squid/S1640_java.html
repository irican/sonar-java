当映射（Map）的所有键（key）都是来自同一枚举的值时，Map 可以用 EnumMap替换，这可能比其他集合更有效， 因为底层数据结构是一个简单的数组。

不合规的代码示例

public class MyClass {

  public enum COLOR {
    RED, GREEN, BLUE, ORANGE;
  }

  public void mapMood() {
    Map<COLOR, String> moodMap = new HashMap<COLOR, String> ();
  }
}

合规的解决方案

public class MyClass {

  public enum COLOR {
    RED, GREEN, BLUE, ORANGE;
  }

  public void mapMood() {
    EnumMap<COLOR, String> moodMap = new EnumMap<> (COLOR.class);
  }
}
