无限循环是一个在程序运行时永远不会结束的循环，即你必须终止程序才能退出循环。无论是通过满足循环结束条件还是通过 break ，每个循环都应该具有结束条件。

不符合规范的代码示例

for (;;) {  // 不符合要求的; 结束条件省略
  // ...
}

int j;
while (true) { // 不符合要求的; 结束条件省略
  j++;
}

int k;
boolean b = true;
while (b) { // 不符合要求的; b从未写入循环
  k++;
}

Compliant Solution

int j;
while (true) { // 可达到的最终条件已添加
  j++;
  if (j  == Integer.MIN_VALUE) {  // 如果值为Integer.MAX_VALUE +1，则为真
    break;
  }
}

int k;
boolean b = true;
while (b) {
  k++;
  b = k < Integer.MAX_VALUE;
}
