将所有分支置于 switch 或 if 链中具有相同的实现是错误的。 要么发生了复制粘贴错误，要么应该执行不同的操作，或者根本不应该有 switch / if 链。

不合规的代码示例

if (b == 0) {  // 不合规的
  doOneMoreThing();
} else {
  doOneMoreThing();
}

int b = a > 12 ? 4 : 4;  // 不合规的

switch (i) {  // 不合规的
  case 1:
    doSomething();
    break;
  case 2:
    doSomething();
    break;
  case 3:
    doSomething();
    break;
  default:
    doSomething();
}

例外

此规则不适用于如果链没有 else -s，或 switch -es没有 default 条款。

if(b == 0) {    //没问题，这可能是为了使代码更具可读性
  doSomething();
} else if(b == 1) {
  doSomething();
}
