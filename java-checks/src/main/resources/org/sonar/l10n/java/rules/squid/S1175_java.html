垃圾收集器在对象变为未引用后的某个时刻调用Object.finalize() 。

通常，重载 Object.finalize() 不是个好主意，因为：

  • 垃圾收集器可能无法调用重载。
  • 用户不应该调用 Object.finalize() ，这会让用户感到困惑。

但是除此之外，如果方法实际上没有重写 Object.finalize()，那么将其命名为“finalize”是一个糟糕的主意。

不合规的代码示例

public int finalize(int someParameter) {        // 不合规
  /* ... */
}

合规的解决方案

public int someBetterName(int someParameter) {  // 合规
  /* ... */
}
