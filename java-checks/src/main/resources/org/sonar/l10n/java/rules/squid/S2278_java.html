根据美国国家标准与技术研究院（NIST）的数据，数据加密标准（DES）不再被认为是安全的：/p>

1977年联邦机构采用保护敏感，非机密信息的方式，DES正在撤销，因为它不再提供保护联邦政府信息所需的安全保障。

鼓励联邦机构使用高级加密标准，这是一种在2001年被批准为FIPS 197的更快更强的算法。

出于类似的原因，也应避免使用RC2。

不合规的代码示例

Cipher c = Cipher.getInstance("DESede/ECB/PKCS5Padding");

合规解决方案

Cipher c = Cipher.getInstance("AES/GCM/NoPadding");
