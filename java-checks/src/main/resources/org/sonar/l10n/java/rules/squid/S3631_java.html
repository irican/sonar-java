对于对象数组， Arrays.asList（T ... a）.stream（）和 Arrays.stream（array）在性能方面基本相同。 但是，对于基元数组，使用 Arrays.asList 将强制构造一个盒装类型列表，然后使用that列表作为流。 另一方面， Arrays.stream 使用适当的基本流类型（ IntStream ， LongStream ， DoubleStream ） 适用，性能更好。

不合规的代码示例

Arrays.asList("a1", "a2", "b1", "c2", "c1").stream()
    .filter(...)
    .forEach(...);

Arrays.asList(1, 2, 3, 4).stream() // 不合规的
    .filter(...)
    .forEach(...);

合规的解决方案

Arrays.asList("a1", "a2", "b1", "c2", "c1").stream()
    .filter(...)
    .forEach(...);

int[] intArray = new int[]{1, 2, 3, 4};
Arrays.stream(intArray)
    .filter(...)
    .forEach(...);
