正则表达式功能强大，但很棘手，即使是那些长期使用正则表达式的人也可能会犯错误。

下列内容不应用作正则表达式：

  • . - 匹配任何单个字符。在 replaceAll中使用，它匹配所有字符
  • | - 通常用作选项分隔符。使用独立，它匹配字符之间的空间
  • File.separator - 匹配特定于平台的文件路径分隔符。在Windows上，这将作为转义字符

不合规的代码示例

String str = "/File|Name.txt";

String clean = str.replaceAll(".",""); // 不合规； 可能只删除点字符，但返回一个空字符串
String clean2 = str.replaceAll("|","_"); // 不合规； 产物 _/_F_i_l_e_|_N_a_m_e_._t_x_t_
String clean3 = str.replaceAll(File.separator,""); // 不合规； Windows上的异常

String clean4 = str.replaceFirst(".",""); // 不合规；
String clean5 = str.replaceFirst("|","_"); // 不合规；
String clean6 = str.replaceFirst(File.separator,""); // 不合规；
