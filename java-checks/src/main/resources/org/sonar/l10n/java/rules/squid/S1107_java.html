共享编码约定使团队可以高效协作。

此规则使得必须将后花括号放在与下一个 else， catch 或 finally 关键字相同的行上。

不合规的代码示例

public void myMethod() {
  if(something) {
    executeTask();
  } else if (somethingElse) {
    doSomethingElse();
  }
  else {                               // 不合规
     generateError();
  }

  try {
    generateOrder();
  } catch (Exception e) {
    log(e);
  }
  finally {                            // 不合规
    closeConnection();
  }
}

合规的解决方案

public void myMethod() {
  if(something) {
    executeTask();
  } else if (somethingElse) {
    doSomethingElse();
  } else {
     generateError();
  }

  try {
    generateOrder();
  } catch (Exception e) {
    log(e);
  } finally {
    closeConnection();
  }
}
