应该使用适当类型的多个catch块，而不是捕获一般异常，然后测试类型。

不合规的代码示例

try {
  /* ... */
} catch (Exception e) {
  if(e instanceof IOException) { /* ... */ }         // 不合规
  if(e instanceof NullPointerException{ /* ... */ }  // 不合规
}

Compliant Solution

try {
  /* ... */
} catch (IOException e) { /* ... */ }                // 合规
} catch (NullPointerException e) { /* ... */ }       // 合规

请参阅

  • CERT, ERR51-J. - Prefer user-defined exceptions over more general exception types
