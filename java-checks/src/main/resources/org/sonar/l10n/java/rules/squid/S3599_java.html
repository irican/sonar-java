因为Double Brace Initialization（DBI）创建了一个匿名类，其引用了拥有对象的实例，如果匿名内部类被其他对象返回并保持，则它的使用可能导致内存泄漏。 即使没有泄漏，DBI也是如此模糊，以至于它会让大多数维护者感到困惑。

对于集合，请改用 Arrays.asList ，或者将每个项目直接添加到集合中。

不合规的代码示例

Map source = new HashMap(){{ //不合规的
    put("firstName", "John");
    put("lastName", "Smith");
}};

合规的解决方案

Map source = new HashMap();
// ...
source.put("firstName", "John");
source.put("lastName", "Smith");
// ...
