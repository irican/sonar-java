使用非标准算法很危险，因为确定的攻击者可能会破坏算法并破坏受保护的数据。 应该使用标准算法，如SHA-256, SHA-384, SHA-512, ...

此规则跟踪java.security.MessageDigest 子类的创建。

推荐的安全编码惯例

  • 使用标准算法而不是创建自定义算法

不合规的代码示例

MyCryptographicAlgorithm extends MessageDigest {
  ...
}
