根据“共同弱点枚举”（Common Weakness Enumeration）

如果两个验证表单具有相同的名称，Struts验证器（Struts Validator）会任意选择其中一个表单用于输入验证并丢弃另一个表单。这个决定可能与程序员的期望不一致......

在这种情况下，很可能应该组合两种形式。至少应该删除一个。

不符合规范的代码示例

<form-validation>
  <formset>
    <form name="BookForm"> ... </form>
    <form name="BookForm"> ... </form>  <!-- 不合规的 -->
  </formset>
</form-validation>

合规的解决方案

<form-validation>
  <formset>
    <form name="BookForm"> ... </form>
  </formset>
</form-validation>

  • OWASP, Improper Data Validation - Struts: Duplicate Validation Forms
