当两种方法具有相同的实现时，要么是错误的 - 还是其他意图 - 或者重复是故意的，但可能会让维护者感到困惑。 在后一种情况下，一个实现应该调用另一个。 不考虑数字和字符串文字。

不符合规范的代码示例

private final static String CODE = "bounteous";

public String calculateCode() {
  doTheThing();
  return CODE;
}

public String getName() {  // 不合规的
  doTheThing();
  return CODE;
}

合规的代码示例

private final static String CODE = "bounteous";

public String getCode() {
  doTheThing();
  return CODE;
}

public String getName() {
  return getCode();
}

例外

忽略少于2个语句的非访问器（getter和setter）的方法。
