“secure”属性阻止cookie通过明文连接(如HTTP)发送，因为它们很容易被窃听。相反，带有secure属性的cookie只通过加密的HTTPS连接发送。

推荐的安全编码实践

  • 在 Cookie 对象上调用 setSecure(true)

不合规的代码示例

Cookie c = new Cookie(SECRET, secret);  // 不合规； cookie 不安全
response.addCookie(c);

合规的解决方案

Cookie c = new Cookie(SECRET, secret);
c.setSecure(true);
response.addCookie(c);
