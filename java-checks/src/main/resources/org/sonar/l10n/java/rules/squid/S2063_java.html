非序列化的 Comparator 可以防止其他 Serializable 有序集合被序列化。 由于使 Comparator 序列化的开销通常较低， 因此这样做可以被认为是良好的防御性编程。

不合规的代码示例

public class FruitComparator implements Comparator<Fruit> {  // 不合规
  int compare(Fruit f1, Fruit f2) {...}
  boolean equals(Object obj) {...}
}

合规的解决方案

public class FruitComparator implements Comparator<Fruit>, Serializable {
  private static final long serialVersionUID = 1;

  int compare(Fruit f1, Fruit f2) {...}
  boolean equals(Object obj) {...}
}
