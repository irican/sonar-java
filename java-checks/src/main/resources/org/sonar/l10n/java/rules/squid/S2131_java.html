“Boxing”是将原始类型值放入原始类型包装器（primitive-wrapper）对象的过程。当纯粹使用包装器类 toString方法时，这会浪费内存和周期， 因为这些方法是 static的，因此可以在没有类实例的情况下使用。类似地，应该避免在具有非String 参数的原始类型包装器类中使用 static 方法 valueOf 。

不合规的代码示例

int myInt = 4;
String myIntString = (new Integer(myInt)).toString(); // 不合规； 创建 &； 弃用了一个 Integer 对象
myIntString = Integer.valueOf(myInt).toString(); // 不合规

合规的解决方案

int myInt = 4;
String myIntString = Integer.toString(myInt);
