命名良好的函数可以让代码的用户在阅读文档之前，一眼就能理解该函数的功能。为此，返回布尔值的方法的名称应该以“is”或“has”开头，而不是以“get”开头。

不合规的代码示例

public boolean getFoo() { // 不合规
  // ...
}

public boolean getBar(Bar c) { // 不合规
  // ...
}

public boolean testForBar(Bar c) { // 合规 - 该方法不以'get'开头。
  // ...
}

合规的解决方案

public boolean isFoo() {
  // ...
}

public boolean hasBar(Bar c) {
  // ...
}

public boolean testForBar(Bar c) {
  // ...
}

例外

不包括重写（override）方法

@Override
public boolean getFoo(){
  // ...
}
