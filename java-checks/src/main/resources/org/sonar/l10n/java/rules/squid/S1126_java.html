应该简化包含在 if-then-else 中的布尔文字语句的返回值。

类似地，包含在 if-then-else 中的方法调用（invocation）仅与布尔文字不同，应简化为单个调用。

不合规的代码示例

boolean foo(Object param) {
  if (expression) { // 不合规
    bar(param, true, "qix");
  } else {
    bar(param, false, "qix");
  }

  if (expression) {  // 不合规
    return true;
  } else {
    return false;
  }
}

合规的解决方案

boolean foo(Object param) {
  bar(param, expression, "qix");

  return expression;
}
