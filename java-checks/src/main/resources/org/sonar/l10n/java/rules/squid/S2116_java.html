虽然数组上可以使用 hashCode 和 toString ，但它们在很大程度上是无用的。 hashCode 返回数组的“标识哈希码（identity hash code）”， toString 返回几乎相同的值。 这两种方法的输出都没有实际反映数组的内容。 相反，您应该将数组传递给相关的静态 Arrays 方法。

不合规的代码示例

public static void main( String[] args )
{
    String argStr = args.toString(); // 不合规
    int argHash = args.hashCode(); // 不合规

合规的解决方案

public static void main( String[] args )
{
    String argStr = Arrays.toString(args);
    int argHash = Arrays.hashCode(args);
