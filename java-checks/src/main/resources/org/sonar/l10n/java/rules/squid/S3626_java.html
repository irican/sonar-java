诸如 return 和 continue 之类的跳转语句允许您更改程序执行的默认流程，但是将控制流程指向原始方向的跳转语句只是浪费 击键。

不符合规范的代码示例

public void foo() {
  while (condition1) {
    if (condition2) {
      continue; // 不合规的
    } else {
      doTheThing();
    }
  }
  return; // 不合规; 这是一种无效方法
}

合规解决方案

public void foo() {
  while (condition1) {
    if (!condition2) {
      doTheThing();
    }
  }
}
