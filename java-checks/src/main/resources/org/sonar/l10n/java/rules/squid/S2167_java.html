重要的是符号，而不是 compareTo 返回的值的大小（magnitude）。 返回 Integer.MIN_VALUE 使 not 传达更高的不等式等级， 这样做会导致错误，因为 compareTo 的返回值有时会期望负值变为正值，从而被反转 。但是，反转 Integer.MIN_VALUE会产生 Integer.MIN_VALUE 而不是 Integer.MAX_VALUE。

不合规的代码示例

public int compareTo(MyClass) {
  if (condition) {
    return Integer.MIN_VALUE;  // 不合规的 
  }

合规的解决方案

public int compareTo(MyClass) {
  if (condition) {
    return -1;
  }
