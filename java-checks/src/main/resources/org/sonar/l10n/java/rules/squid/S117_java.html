共享命名约定能使团队有效协作。当局部变量或函数参数名称与提供的正则表达式不匹配时，此规则会提出问题。

不合规的代码示例

使用默认的正则表达式 ^[a-z][a-zA-Z0-9]*$：

public void doSomething(int my_param) {
  int LOCAL;
  ...
}

合规的解决方案

public void doSomething(int myParam) {
  int local;
  ...
}

例外

此规则将忽略循环计数器。

for (int i_1 = 0; i_1 < limit; i_1++) {  // 合规
  // ...
}

以及单字符 catch 变量： 

try {
//...
} catch (Exception e) { // 合规
}
