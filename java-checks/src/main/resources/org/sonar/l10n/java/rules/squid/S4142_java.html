有一些有效的例子可以将变量多次传递到同一个方法调用中，但通常这样做是错误的，而其他一些参数也是如此。

不合规的代码示例

if (compare(myPoint.x, myPoint.x) != 0) { // 不合规的
  //...
}

if (compare(getNextValue(), getNextValue()) != 0) { // 不合规的
  // ...
}

合规解决方案

if (compare(myPoint.x, myPoint.y) != 0) {
  //...
}

Object v1 = getNextValue();
Object v2 = getNextValue();
if (compare(v1, v2) != 0) {
  // ...
}
