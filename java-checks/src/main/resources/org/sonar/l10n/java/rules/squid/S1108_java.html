共享编码约定使团队可以高效协作。

此规则强制要求将后花括号和下一个 else， catch 或 finally 关键字放在不同的两行。

不合规的代码示例

public void myMethod() {
  if(something) {
    executeTask();
  } else if (somethingElse) {          // 不合规
    doSomethingElse();
  }
  else {                               // 合规
     generateError();
  }

  try {
    generateOrder();
  } catch (Exception e) {
    log(e);
  }
  finally {
    closeConnection();
  }
}

合规的解决方案

public void myMethod() {
  if(something) {
    executeTask();
  }
  else if (somethingElse) {
    doSomethingElse();
  }
  else {
     generateError();
  }

  try {
    generateOrder();
  }
  catch (Exception e) {
    log(e);
  }
  finally {
    closeConnection();
  }
}
