在没有括号花括号的情况下，紧跟在条件之后的行是有条件执行的行。 通过惯例和良好实践，这些线是缩进的。 在没有花括号和缩进的情况下，原始程序员的意图完全不清楚，也许实际上并不是执行的内容。 此外，这样的代码很可能会让维护者感到困惑。

不符合规范的代码示例

if (condition)  // 不符合规范的
doTheThing();

doTheOtherThing();
somethingElseEntirely();

foo();

合规的解决方案

if (condition)
  doTheThing();

doTheOtherThing();
somethingElseEntirely();

foo();
